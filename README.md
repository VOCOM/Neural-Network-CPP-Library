# RSE4207
Artificial Intelligence model
Language: `C++`\
Build system: `CMake`\
Supported Operating Systems: `Windows, Linux`

How to use:
1. Clone the repository
2. Open a terminal in the repository
3. ```
   mkdir build && cd build
   cmake ..
   ```
For Command Prompt:
```
msbuild.exe RSE4207_AI_MODEL.sln
```

For Bash:
```
make
./bin/RSE4207_AI_MODEL

```
